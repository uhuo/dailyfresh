<p>dailyfresh mall based on B2C model<br>
基于B2C的天天生鲜商城</p>
<blockquote>
<p>B2C(Business-to-Customer), 企业对个人的一种商业模式，简称"商对客". 商对客是电子商务的一种模式，这种电子商务一般以网络零售业为主，主要借助于互联网开展在线销售活动。 B2C即企业通过互联网为消费者提供一个新型的购物环境——网上商店，消费者通过网络在网上购物、网上支付等消费行为。</p>
</blockquote>
<p>为消费者提供一个新型的购物环境:<br>
从0开始，立项、构建、开发到部署, 搭建一个天天生鲜商场</p>

<li>页面数据缓存，把页面使用的数据存放在缓存中，当再次使用这些数据时，先从缓存中获取，如获取不到，再去查询数据库，减少数据库的查询次数。</li>
